All aboard, dear passengers,


We're very excited to announce that the release date for Re:Turn 2 - Runaway has been set and will be available at the Nintendo Switch, Xbox and Steam on the 28th January 2022. Wishlist now and stay tuned for more info soon!

If you haven’t taken this train before, we highly recommend that you begin your journey with Re:Turn - One Way Trip, the first game where it all began…

It’s the perfect time to experience it from the beginning while the next instalment creeps ever closer.

We hope to see you again soon. Perhaps on January 28th?
